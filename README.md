# 🏃‍♂️ Nuxt-runner Monorepo

## 🎯 О проекте

Nuxt-runner — это приложение для отслеживания и анализа беговых (и не только) тренировок. Здесь в скором времени будут представлены инструменты для записи тренировок, просмотра подробной статистики, личных рекордов, достижений и многого другого. Проект представляет собой монорепозиторий, в котором также планируется разработка API и мобильного приложение.

---

## 🚀 Стек

- **Монорепозиторий:** `pnpm workspaces`
- **Фронтенд (Web):** `Nuxt 4` (Vue 3, TypeScript)
- **Бэкенд (API):** `NestJS` (TypeScript, Node.js)
- **База данных:** `PostgreSQL || (&&) Firebase`
- **Общие типы:** `TypeScript` (`shared-types` package)
- **Мобильное приложение:** `Flutter` (Dart)
