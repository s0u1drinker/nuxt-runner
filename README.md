# 🏃‍♂️ Nuxt-runner Monorepo

## 🎯 О проекте

Nuxt-runner — это приложение для отслеживания и анализа беговых (и не только) тренировок. Здесь в скором времени будут представлены инструменты для записи тренировок, просмотра подробной статистики, личных рекордов, достижений и многого другого.

---

## 🚀 Стек

- **Монорепозиторий:** `pnpm workspaces`
- **Фронтенд (Web):** `Nuxt 4` (TypeScript)
- **Бэкенд (API):** `NestJS` (TypeScript)
- **База данных:** `PostgreSQL`
- **Мобильное приложение:** `Flutter`
